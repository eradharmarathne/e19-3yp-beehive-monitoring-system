Beehive Monitoring System API

This API allows interaction with beehive data, camera records, and user information for the Beehive Monitoring System.

User Endpoints

POST /api/user/register

Register a new user. Send user details in the request body.

curl -X POST http://beehive-backend.eba-xkm8nww8.us-west-2.elasticbeanstalk.com/api/user/register -d '{"email":"example@beezee.com", "username": "user", "password": "Pass123!"}'

POST /api/user/login

Login for existing users. Returns a token on successful authentication.

curl -X POST http://beehive-backend.eba-xkm8nww8.us-west-2.elasticbeanstalk.com/api/user/login -d '{"email":"example@beezee.com", "password": "Pass123!"}'

GET /api/user/current

Get current user details. Requires a valid token.

curl -X GET http://beehive-backend.eba-xkm8nww8.us-west-2.elasticbeanstalk.com/api/user/current -H "Authorization: Bearer YourToken"

Beehive Endpoints

GET /api/beehive

Retrieve all beehives. Requires a valid token.

curl -X GET http://beehive-backend.eba-xkm8nww8.us-west-2.elasticbeanstalk.com/api/beehive -H "Authorization: Bearer YourToken"

POST /api/beehive

Create a new beehive record. Requires a valid token.

curl -X POST http://beehive-backend.eba-xkm8nww8.us-west-2.elasticbeanstalk.com/api/beehive -d '{"location": "location1", "type": "type1"}' -H "Authorization: Bearer YourToken"

PUT /api/beehive/{id}

Update an existing beehive record by ID. Requires a valid token and beehive ID.

curl -X PUT http://beehive-backend.eba-xkm8nww8.us-west-2.elasticbeanstalk.com/api/beehive/{id} -d '{"location": "newLocation", "type": "newType"}' -H "Authorization: Bearer YourToken"

DELETE /api/beehive/{id}

Delete an existing beehive record by ID. Requires a valid token and beehive ID.

curl -X DELETE http://beehive-backend.eba-xkm8nww8.us-west-2.elasticbeanstalk.com/api/beehive/{id} -H "Authorization: Bearer YourToken"

GET /api/beehive/{id}

Retrieve details of a specific beehive by ID. Requires a valid token and beehive ID.

curl -X GET http://beehive-backend.eba-xkm8nww8.us-west-2.elasticbeanstalk.com/api/beehive/{id} -H "Authorization: Bearer YourToken"

Camera Record Endpoints

GET /api/camera

Retrieve all camera records. Requires a valid token.

curl -X GET http://beehive-backend.eba-xkm8nww8.us-west-2.elasticbeanstalk.com/api/camera -H "Authorization: Bearer YourToken"

POST /api/camera

Create a new camera record. Requires a valid token.

curl -X POST http://beehive-backend.eba-xkm8nww8.us-west-2.elasticbeanstalk.com/api/camera -d '{"record": "recordData"}' -H "Authorization: Bearer YourToken"

PUT /api/camera/{recordId}

Update an existing camera record by record ID. Requires a valid token and record ID.

curl -X PUT http://beehive-backend.eba-xkm8nww8.us-west-2.elasticbeanstalk.com/api/camera/{recordId} -d '{"newRecordData": "data"}' -H "Authorization: Bearer YourToken"

DELETE /api/camera/{recordId}

Delete an existing camera record by record ID. Requires a valid token and record ID.

curl -X DELETE http://beehive-backend.eba-xkm8nww8.us-west-2.elasticbeanstalk.com/api/camera/{recordId} -H "Authorization: Bearer YourToken"

GET /api/camera/{recordId}

Retrieve details of a specific camera record by record ID. Requires a valid token and record ID.

curl -X GET http://beehive-backend.eba-xkm8nww8.us-west-2.elasticbeanstalk.com/api/camera/{recordId} -H "Authorization: Bearer YourToken"
